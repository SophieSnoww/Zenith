### Question 8

Find the exact values of all trigonometric ratios of $\theta$ in the triangle.

$$ a^2 + 7^2 = 8^2 $$
$$ a^2= 8^2 - 7^2 = 15 $$
$$ a = \sqrt{15} $$

---

$$ \sin{ \theta } = \frac{ 7 }{ 8 } $$
$$ \cos{ \theta } = \frac{ \sqrt{ 15 } }{ 8 } $$
$$ \tan{ \theta } = \frac{ 7 }{ \sqrt{ 15 } } $$
$$ \csc{ \theta } = \frac{ 8 }{ 7 } $$
$$ \sec{ \theta } = \frac{ 8 }{ \sqrt{ 15 } } $$
$$ \cot{ \theta } = \frac{ \sqrt{ 15 } }{ 7 } $$
